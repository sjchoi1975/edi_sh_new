이용자 실적 입력 프로세스 요약

1. 정산월/기간 선택
이용자가 “실적 관리 > 실적 등록” 메뉴 진입
관리자가 등록한 정산월 중 “활성”이고 “실적입력기간”에 해당하는 정산월만 노출
해당 조건이 없으면 “실적입력기간이 아닙니다” 안내 문구 표시

2. 기본 정보 표시
선택된 정산월, 실적입력기간, 기준월 등 기본 정보 표시

3. 실적 입력 대상 거래처/제품 목록
거래처별로 입력 대상 제품 목록이 표로 나옴
각 행에 제품명, 코드, 약가, 처방수량, 처방액, 비고 등 표시
“입력” 버튼 클릭 시 상세 입력 팝업/행 활성화

4. 입력 방식
처방수량, 처방액 등 직접 입력
처방수량 입력 시 처방액 자동 계산(약가 × 수량)
비고는 선택(드롭다운) 또는 직접 입력 가능

5. 저장/등록
입력 완료 후 “등록” 버튼 클릭
저장 성공 시 목록에 반영, 실패 시 에러 안내

6. 기타
이미 등록된 실적이 있으면 수정/삭제 가능
실적입력기간이 지나면 입력/수정 불가, 단순 조회만 가능

정리

실적입력기간 내 활성 정산월만 입력 가능
거래처별, 제품별로 실적 입력
입력/수정/삭제/저장 기능
